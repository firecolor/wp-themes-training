## ファイル構成
*がついているものは今回は直接関係がありません
```
.
├── _complete : 参考用に作成したファイルを格納
│
├── assets : アセットフォルダ
│   ├── css
│   ├── images
│   └── js
│
├── src(*) ： ビルド前のソース
│   ├── scss
│   └── js
│
├── functions.php
├── style.css
├── style.css.map(*)
├── screenshot.png
├── LICENSE(*)
│
├── sandbox.php
│
├── package.json(*)
├── package-lock.json(*)
├── webpack.config.js(*)
├── .gitignore(*)
│
├── README.md(*)

```
