# WordPressテーマ作成の練習用

WordPressのテーマ講座用の教材になります。

----

## ファイル構成
```
.
├── _complete : 参考用に作成したファイルを格納
│
├── assets : アセットフォルダ
│   ├── css
│   │
│   ├── images
│   │
│   └── js
│       └── app.js : 配信用のバンドルファイル。JSの別ファイルにする場合はビルド設定を書き換え
│
├── src ： ビルド前のソース
│   ├── scss
│   │
│   └── js
│
├── template-parts
│
├── functions.php
├── index.php
├── style.css
├── screenshot.png
├── LICENSE
│
├── sandbox.php
│
├── package.json : パッケージファイル
├── package.json : パッケージファイル
├── package-lock.json
├── webpack.config.js : webpackの設定
│
├── README.md :         このファイル

```
